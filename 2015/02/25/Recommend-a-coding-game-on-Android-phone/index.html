Recommend a coding game on Android phone

Feb 25 2015

2 days ago, I found a game on the Android OS called hacked. I find it highly entertaining so I’d like to intro it to you.

hacked is so-called world’s first mobile coding game, and it is in BETA period. So it seems that we cannot install it from Google Play Store. I down it from a 3rd-party store. I am not sure if it’s ok to do that obeying the rules. Anyway, here some info for you:

  • hacked uses a simple and powerful language to code, whose name is H. It’s simple for the documentation is so short. But there have been a lot of little game coded in H language.
  • hacked has a UI which take advantage of the touchscreen. Although I did not use it properly before I watched its video, it is designed for phone on purpose. Instead of letting user type the names of vars in, hacked is designed to use a flexible keyboard which changes with the position of cursor with var-name key like var_a or f1. It’s not a good way to name you varibles but on phone it’s good enough.
  • The story mode contains a heroic story, about FBI things.

You may think that ‘I am not a coder, why this game?’. Maybe you are right. But, it’s a game! Is is not enough for you to play it? You play sudoku, you play puzzles, you play Rubik Cubes, here is a new option for you intelligent guys. Just try it. And I think it’s a better choice for those who want to learn programming but are non-professional programmer to kick start.
