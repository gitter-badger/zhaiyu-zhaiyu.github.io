Compile Emacs

Jun 19 2015

Well, actually, I am a Vimmer.

But Prof. Nooijen from University of Waterloo visits us these days and he says he can only use Emacs.

Well…

If your server is not connect to the Internet, you cannot simply use sudo apt-get install emacs. Or sudo zypper install emacs if you are a SUSE user.

First thing first you should compile one from source code. The most important thing to do is configure.

For those who do calculation via CLI (like us), you should cd to the emacs source directory and type in ./configure --without-all --without-x.

Then you can make and make install.

I think installation of Emacs is much easier than that of Vim. Things above is what you should do if you are new guy.

Remember chmod 777 those scripts according to the error message.
